1.6 - CSS Básico Introdução ao CSS O CSS Cascading Style Sheets, ou folhas de estilo em cascata, é a tecnologia que usamos para adicionar estilo ao HTML. Com o CSS, podemos deixar as páginas da web mais atraentes, definindo cores, tamanhos, espaçamentos e outros aspectos visuais. Se o HTML é o esqueleto da web, o CSS é a "roupa" que a deixa bonita. Durante este módulo, você aprenderá as três principais formas de aplicar CSS ao HTML e como utilizá-las de maneira eficiente em seus projetos. Anatomia de uma Declaração CSS Uma declaração CSS consiste de seletores e propriedades. Um seletor é o elemento HTML ao qual queremos aplicar o estilo, e a propriedade é o que será modificado. Aqui está um exemplo básico: h1 color: blue; font-size: 20px; - h1 é o seletor. - color e font-size são propriedades. - blue e 20px são os valores aplicados às propriedades.
